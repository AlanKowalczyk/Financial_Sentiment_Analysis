# Financial Sentiment Analysis

## Overview  
NLP project extracting sentiment from financial text to analyse market signal.

## Key Features
* **Advanced NLP:** Utilizes `FinBERT` and `Twitter-RoBERTa` for domain-specific sentiment classification.
* **Hyperparameter Optimization:** Implements Bayesian Optimization using **Optuna** to tune XGBoost, LightGBM, and Random Forest classifiers.
* **Parallel Processing:** Optimized data cleaning and inference pipelines using Python's `multiprocessing`.


## Tools & Techniques  
Python, NLTK/spaCy, pandas, Optuna, Scikit-learn, HuggingFace (FinBERT)
